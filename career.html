Software Developer

• Bachelor of Computer Science, Computer Engineering, or related field
• 1+ years of software development experience (Core: Python)
• Should have deep knowledge of Agile development process & should be able to play the role of a Scrum Master / team lead
• Good to Have: Tools experience of Jira, Git
• Good to Have: SQL Queries working knowledge
• Possess strong computer science fundamentals: data structures, algorithms, programming languages, distributed systems, and information retrieval
• Ability to prioritize and multi-task in order to meet deadlines

Employment Type: Permanent Job, Full Time
Key skills: Python, Python Developer, python programming, Django

Embedded software engineer

• 0 to 1 years of experience in Embedded C
• Hands on development experience on UART, I2C, SPI programming in 8051, AVR, ESP8266 SOC, Arm cortex family controller
• Hands on development experience in python on Linux based Board Raspberry Pi, Beagle Bone
• Knowledge of Networking topology
• Communication protocols like MQTT, HTTP, Socket programming TCPIP, UDP

Employment Type: Permanent Job, Full Time
Keys kills: microcontroller, microprocessor, raspberry pi , arduino, Nodemcu , Embedded C

3D Modeller(Design Engineer)

• must be a Graduate / Diploma holder
• Special course for AutoCAD/ Unigraphics / Solid works / Pro-e. Technical / Professional
• Should have knowledge of Unigraphics / solid works, / AutoCAD / Pro-e
• Should be able to work on product design for plastic products in 2D or 3D & modification.
• Should have worked on such field at least 1 years or more
• Should be comfortable with Unigraphics / AutoCAD / Pro-e / Solid works.

Employment Type: Permanent Job, Full Time
Key skills: product design, 2d, 3d plastic, plastics products, mould design, 3d modeling, 2d modeling, AutoCAD, Unigraphics, CAD, CAM

Holographic 3D Designer

• Excellent knowledge of Autodesk tools, in particular 3D Studio Max.
• Good all-round knowledge of modelling, rigging, layout, animation, colouring, texture predominantly for non-organic designs.
• Excellent rendering, lighting and shading skills.
• Good knowledge of programming and scripting e.g. MaxScript or Python.
• Working knowledge of other 3D design tools e.g. Maya.
• Familiarity with popular 3D file formats e.g. OBJ, point cloud data.
• Video animation

Employment Type: Permanent Job, Full Time
Key skills: product design, 2d, 3d, MAYA, CAD, CAM
